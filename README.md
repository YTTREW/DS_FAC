### Ejercicio 3

Para poder instalar las dependencias necesarias recomendamos el uso de un entorno virtual.

```
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Para ejecutar el programa:

```
python3 ejercicio3.py
```
